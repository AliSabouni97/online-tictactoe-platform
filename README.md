# Online Tic-Tac-Toe Platform

An online turn-based game platform MVP built to demonstrate clean project structure, server-authoritative game design, and real-time communication using modern web technologies.

## Tech Stack

**Frontend**

- React
- TypeScript
- Vite
- TailwindCSS

**Backend**

- Node.js
- TypeScript
- Express
- Socket.IO

**Tooling**

- ESLint
- Prettier
- Vitest

## Project Structure

```
online-tictactoe-platform/
├── client/        # Frontend application
├── server/        # Backend server
├── shared/        # Shared types and constants
└── README.md
```
